# Duoloot Red Vault Icon Library

Pacote de ícones SVG personalizados para o site Duoloot.

## Conteúdo
- 35+ ícones SVG individuais
- `sprite.svg` para uso como sprite sheet
- `manifest.json` com categorias
- `preview.html` com galeria visual

## Exemplo de uso (img)
```html
<img src="./vault.svg" alt="Vault" width="24" height="24" />
```

## Exemplo de uso (sprite)
```html
<svg width="24" height="24" aria-hidden="true">
  <use href="./sprite.svg#vault"></use>
</svg>
```
