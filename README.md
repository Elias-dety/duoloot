# 🎮 Duo Loot - Gamified eSports Experience

O **Duo Loot** é uma plataforma de última geração desenvolvida para conectar jogadores de eSports, oferecendo um ecossistema completo de lobbies competitivos, serviços de coaching de alto nível e um sistema de recompensas gamificado (Vault).

![Duo Loot](https://images.unsplash.com/photo-1542751371-adc38448a05e?auto=format&fit=crop&q=80&w=1200)

## 🚀 Status do Projeto: Sprint 1 Finalizada

A primeira fase do projeto foi concluída com sucesso, estabelecendo a fundação arquitetural e as principais interfaces de usuário utilizando **Atomic Design** e **Design Tokens**.

### Principais Funcionalidades Implementadas:
- **Dashboard Holístico:** Visão 360º do progresso do jogador.
- **Marketplace de Coaches:** Filtros avançados e cards detalhados.
- **Sistema Premium:** Landing page de conversão com comparativo de planos.
- **Vault (Cofre):** Experiência de abertura de loot boxes e progresso.
- **Player Profile:** Identidade visual e estatísticas integradas.
- **Lobby System:** Listagem dinâmica de salas competitivas.

## 🛠️ Stack Tecnológica

- **Frontend:** [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) + CSS Variables
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Validação:** [Zod](https://zod.dev/)
- **Roteamento:** [React Router 7](https://reactrouter.com/)

## 🏗️ Arquitetura Atômica

O projeto segue rigorosamente o padrão **Atomic Design**, garantindo escalabilidade e reutilização:

- **Atoms:** Componentes indivisíveis (Botões, inputs, badges).
- **Molecules:** Grupos de átomos (Stat cards, timers).
- **Organisms:** Seções complexas (Hero sections, grids).
- **Templates:** Layouts de página.
- **Pages:** Páginas finais com injeção de dados.
