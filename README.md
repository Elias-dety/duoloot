# Duo Loot

Duo Loot é uma plataforma gamer criada para conectar jogadores, formar duos melhores e organizar experiências competitivas com mais confiança.

O foco do projeto não é apenas mostrar perfis bonitos. O foco é ajudar jogadores a encontrar parceiros compatíveis, entrar em lobbies úteis, validar identidade gamer quando necessário e evoluir a experiência com recursos como matchmaking, Vault/Cofre, Riot/VALORANT e planos Premium.
